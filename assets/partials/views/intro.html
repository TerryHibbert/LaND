Why is it called LaND?

LaND is an abbreviation of "Layout Numerator / Denominator" which describes how the system works and is the format of CSS classes applied to HTML elements to implement the system.

i.e. The Layout part denotes a browser width range to target. TheNumerator / Denominator part is the fraction of horizontal width of parent container that the column will fill.

How do you use it?

Classes are added to an element. The actual syntax is this:

<div class="col s-1-2">Hello</div>
<div class="col s-1-2">world</div>

The result of two of the above together is this:

Hello world
Hello world

An explanation of the above: The 'col' declares the element to be a LaND column. A land column has a default layout of 't-1-1' which means
"on small screens and wider, the columns has a width of 1 divided by 1 (which is the whole width)". The 't-1-2' specifies that
"for tablet and wider, the columns have a width of 1 divided by 2 (half)".

What's with the colors?

The colors are what the site looks like in debug mode. The purpose of debug mode helps to see where the 'col' blocks start and end. It's enabled through two classes and toggleable with this bookmarklet (LaND Debug), which you can drag to your bookmark bar.
